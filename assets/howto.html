How to use JoyToy

Home




Edit Panel (top-left of the screen)
- write source code in this area.

Console Panel (top-right of the screen)
- result of program displays in this area.(use method 'print')

Buttons (bottom of the screen)
  1. 'Eval' - run the program in the Edit Panel.
  2. 'Load' - display files list for load a program file.
  3. 'Save' - display edit view for save a program.
  4. 'Clear' - clear Console Panel.
  5. 'Help' - display this help.
